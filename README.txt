Commands to run the Code:

1> After cloning the code , Please go to app folder then run npm install
2> Run npm start for the server to start



WebPage Instructions:

1> After you start the server go to webpage localhost:8000 on your browser.
2> Once the page is open  you have two options to choose on your left side for line and Bar Charts.
3> Once you choose the required option then you can select the parameters for y-axis by selecting the required radio buttons.
